Aplicación web para encriptar y desencriptar texto usando un cifrado por sustitución de vocales. Inspirado en el challenge de Alura Latam + Oracle ONE.

**Reglas de encriptación:**
- `e` → `enter`
- `i` → `imes`
- `a` → `ai`
- `o` → `ober`
- `u` → `ufat`

## Características

- Encriptar y desencriptar texto al instante
- Copiar resultado al portapapeles
- Validación: solo minúsculas sin acentos
- Diseño responsive con tema oscuro
- Interfaz limpia con feedback visual

## Stack

- HTML5 semántico con atributos ARIA
- CSS3 con custom properties y diseño responsive (320px-768px+)
- JavaScript vanilla (ES6+)
- SweetAlert2 para notificaciones

## Uso

Abre `index.html` en cualquier navegador moderno (requiere internet para CDN de fuentes y SweetAlert2).
